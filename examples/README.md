Sample input data folder contains files from the dataset. Should the end--user wish to run this on the entire dataset, please look into the documentation on usage of classes in **tools** directory.
call_transi_class.py is a sample script that uses the transition data and transition post processor class from **tools** to perform data extraction and visualization.
call_turb_class.py is a sample script that uses the turbulence data and turbulence post processor class from **tools** to perform data extraction and visualization.
